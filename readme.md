# bsmi_ocaml common library

Document: [https://infobsmi.github.io/bsmi_ocaml/index.html](https://infobsmi.github.io/bsmi_ocaml/index.html)

## build and running  

```bash
 dune build 
# build document
dune build @docopen

# format the source code
dune build @ff

# run example app
 dune exec bsmi_ocaml
```
